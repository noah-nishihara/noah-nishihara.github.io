A small collection of browser-based experiments, simulations, viewers, games, and reference pages hosted with GitHub Pages.

## Featured projects

### 🚆 Adelaide Metro 4000 Class Driver Viewer

A WebGL train viewer/simulator for the Adelaide Metro 4000 Class, with driver-style controls, cab view, camera presets, station scenery, and linked GLB model loading.

- **Open:** https://noah-nishihara.github.io/adelaide-4000-train-driver.html
- **Alternative train viewer:** https://noah-nishihara.github.io/train-viewer.html
- **Older v3 train driver build:** https://noah-nishihara.github.io/adelaide_train_driver_v3.html

### 🌊 Submarine simulations

Interactive underwater submarine viewers/simulators with sea ambience, controls, model loading, and camera tools.

- **Submarine Sea Viewer:** https://noah-nishihara.github.io/submarine-sea-viewer.html
- **Submarine Simulator:** https://noah-nishihara.github.io/submarine-simulator.html

### 🐠 Open Goldfish Pond

A calm WebGL pond scene using goldfish GLB models, natural scenery, smooth swimming, and simple controls.

- **Open:** https://noah-nishihara.github.io/open_goldfish_pond.html

### 🐈 Neko95 Maze

A Windows 95 3D Maze-inspired cat adventure with classic maze controls, charms, enemies, a Smiley exit, and screensaver mode.

- **Open:** https://noah-nishihara.github.io/neko95_maze_v1.html

## Notes

- GitHub Pages paths are case-sensitive. For example, `HANABI.html` must be written in uppercase.
- After pushing changes, GitHub Pages may take a minute or two to refresh.
- Most pages are standalone HTML files and can be opened directly from the public GitHub Pages URLs above.
- Some pages rely on external JavaScript/CDN assets and linked GLB model files, so they work best when served through GitHub Pages rather than opened directly from the local filesystem.
